### セットアップ方法
1. こちらのリポジトリをクローンするだけ。

※ライブラリは全て`Swift Package Manager`を用いて導入しているため、`.xcodeproj`を開くと自動でダウンロードされます。

### 各種実行方法
1. ビルド

`command`+`B`または、アプリメニューの`Product`→`Build`

1. 実行

`command`+`R`または、アプリメニューの`Product`→`Run`

2. テスト

`command`+`U`または、アプリメニューの`Product`→`Test`
